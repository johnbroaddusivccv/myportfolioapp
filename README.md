# JMBIV Portfolio in React
Built with Create React App, uses a few react-components I found on [NPM](https://www.npmjs.com/) to assist with making sections respsonive on a mobile level.
## Install

clone Repository

```
npm install
```

```
npm run start
```
### Deployment

Deploy to Heroku

```
heroku create
```

```
git push heroku master
```
